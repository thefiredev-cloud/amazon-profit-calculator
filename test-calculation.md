# Test Calculation Example

## Test Product: Wireless Bluetooth Headphones

Use these values to test the calculator:

### Input Values:
- **Product Name**: Wireless Bluetooth Headphones
- **Selling Price**: $29.99
- **Category**: Electronics (8% referral fee)
- **Dimensions**: 
  - Length: 8 inches
  - Width: 6 inches  
  - Height: 3 inches

- **Profit Margin**: ~33.8%
- **Monthly Profit**: ~$1,014

### Verdict: ✅ Good Profit
- Profit margin above 15%
- Healthy profit per unit
- Strong monthly profit projection
